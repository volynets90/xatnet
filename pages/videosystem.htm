Системи відео-спостереження

Система відеоспостереження (CCTV) - це комплекс обладнання та програмного забезпечення, призначений для спостереження за територією, діями та ситуацією. Сьогодні це - найбільш затребувана система для охоронних та моніторингових цілей. Розвиток систем відеоспостереження відкриває нові можливості не тільки для фіксації правопорушень, а, що є більш важливим, для їх попередження.
